# Quiz App

Интерактивная платформа для прохождения тестов, переработанная как одностраничное приложение (SPA) на Angular 18. Приложение поддерживает аутентификацию пользователей через JWT-токены, выбор и прохождение тестов с таймером, навигацию по вопросам, просмотр результатов с детальным разбором ответов и автоматическое обновление токенов через HTTP-интерсептор.

## Технологический стек

- Angular 18 (standalone-компоненты)
- TypeScript (строгий режим)
- SCSS
- RxJS
- Angular Material
- Angular ESLint

## Структура проекта

```
src/
├── app/
│   ├── core/              # Сервис авторизации, интерсептор, гард
│   ├── features/          # Страницы: главная, авторизация, тест, результаты
│   └── shared/            # Директивы, типы, сервисы, layout
├── assets/                # Изображения, шрифты
├── environments/          # Конфигурация dev / prod окружений
└── styles/                # Глобальные SCSS-стили
```

## Начало работы

Требуется Node.js 18+ и запущенный backend API на `localhost:3000`.

```bash
npm install
ng serve
```

Сервер разработки будет доступен по адресу `http://localhost:4200`.

Для production-сборки:

```bash
ng build
```

## Линтер

```bash
ng lint
```

## Маршруты приложения

| Маршрут | Описание |
|---------|----------|
| `/` | Главная страница |
| `/signup` | Регистрация |
| `/login` | Вход |
| `/choice` | Выбор теста |
| `/test/:id` | Прохождение теста |
| `/result` | Результаты |
| `/result-details` | Подробные результаты с правильными ответами |
